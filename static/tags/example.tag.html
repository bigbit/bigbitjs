Introduction

BigBit standard is an alternate of IEEE 754 standard and UTF encoding.

Benefits

  • No precision loss: Now 0.3 - 0.1 is always 0.2 not 0.19999999999999998. So if you're working on a science or finance project, switch to it immediately.
  • Biggest number: It can represent any number in the universe
  • Take less memory: Unlike binary16, binary32, .. It doesn't occupy fixed blocks to represent a number. And take lesser memory.
  • Less decision, no confusion: LB encoding covers all UTF formats. And three data types (EB, HB, EHB) covers maximum numeric data types, like: integer, float, double, long, decimal, biginteger etc.
